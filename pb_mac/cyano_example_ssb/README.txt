This folder contains the following files:
    
    1. speciesTree.tre
         Maximum-likelihood phylogeny of 189 cyanobacteria.

    2. Calibration.txt
         Fossil calibrations.
    
    3. Outgroup.txt
         List of taxa in the most basal group of the tree.
    
    4. LSU_SSU.phy
         Concatenated alignment of LSU and SSU rRNA sequences in PHYLIP format.
        
    5. phylobayes.sh
         Script file containing the command-line used to run phylobayes.
